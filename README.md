# jod-demos

Analytics experiments for the [*Jatkuvan oppimisen digitaalinen palvelukokonaisuus (JOD)*](https://okm.fi/hanke?tunnus=OKM069:00/2021) project.

## Data used

* ESCO
* konfo (koulutusinformaatio)
* ePerusteet
* Ammattitieto / Ammatinvalintaohjelma

## Experiments

1. similarity
2. restrictions
3. feedback
4. chatbot
